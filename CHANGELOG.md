# 更新日志

## 🎉 v1.1.0 - 2025-10-09

### 新增功能

#### 🆕 腾讯文档表单支持
- ✅ 新增 `TencentDocsFiller` 填写引擎
- ✅ 自动检测表单类型（腾讯文档/麦客CRM）
- ✅ 智能字段匹配（精确匹配 + 模糊匹配）
- ✅ 腾讯文档专用诊断功能
- ✅ 详细的填写结果反馈

#### 技术改进
- ✅ 支持自动移除 `disabled` 和 `readonly` 属性
- ✅ 完善的事件触发机制（`input`, `change`, `blur`）
- ✅ 异步 JavaScript 执行支持
- ✅ 详细的控制台日志输出

### 文件变更

#### 新增文件
- `core/tencent_docs_filler.py` - 腾讯文档填写引擎
- `TENCENT_DOCS_SUPPORT.md` - 腾讯文档支持文档
- `CHANGELOG.md` - 更新日志（本文件）

#### 修改文件
- `core/__init__.py` - 导出 `TencentDocsFiller`
- `gui/auto_fill_window.py` - 添加表单类型检测和多引擎支持
- `README.md` - 更新文档，添加腾讯文档说明

### 使用示例

```python
# 腾讯文档表单链接示例

赞藏量 → 5000

```
