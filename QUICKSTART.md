# 🚀 快速入门指南

## 5 分钟上手

### Step 1: 安装 (2 分钟)

```bash
cd /Users/chenchen/Desktop/开发项目/auto-form-filler-py

# 方式一：使用安装脚本（推荐）
./install.sh

# 方式二：手动安装
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

### Step 2: 启动 (10 秒)

```bash
# 方式一：使用启动脚本
./run.sh

# 方式二：手动启动
source venv/bin/activate
python main.py
```

### Step 3: 使用 (3 分钟)

#### 第一次使用

程序会自动创建测试数据：

- ✅ 名片 1（11 个配置项）
- ✅ 测试链接（抖音招募表单）

#### 开始填写

1. **点击 "📝 自动填写"**
2. **左侧选择 "📇 名片 1"**
3. **左侧选择 "🔗 抖音招募表单"**
4. **点击 "🚀 开始自动填写"**
5. **等待 2-3 秒，填写完成！**

## 📝 创建自己的名片

### 方法一：通过界面创建

1. 主界面点击 "📇 名片管理"
2. 点击 "➕ 新增名片"
3. 填写信息：

```
名片名称: 我的小红书账号
描述: 用于填写小红书合作表单

配置项:
- 平台: 小红书
- 账号: 小明
- 粉丝: 5万
- 互动率: 8%
- 微信: xiaoming123
- 电话: 13800138000
```

4. 点击 "💾 保存"

### 方法二：复制默认名片修改

1. 在名片管理中点击 "名片 1" 的 "编辑"
2. 修改名称和配置项
3. 另存为新名片

## 🔗 添加常用链接

1. 主界面点击 "🔗 链接管理"
2. 点击 "➕ 新增链接"
3. 填写信息：

```
链接名称: 小红书商务合作表单
URL: https://your-form-url.com
分类: 小红书
状态: 激活
```

4. 点击 "💾 保存"

## 🎯 使用技巧

### 提高匹配成功率

| 字段名示例 | ✅ 推荐        | ❌ 不推荐      |

## ❓ 常见问题速查

### 安装问题

**Q: PyQt6 安装失败？**

```bash
# macOS
brew install qt6

# 或使用国内镜像
pip install PyQt6 -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### 使用问题

**Q: 填写失败？**

- 检查字段名是否准确
- 尝试简化字段名（如 "微信" 而不是 "微信号码"）
- 查看右侧配置面板手动填写

**Q: 网页加载慢？**

- 检查网络连接
- 点击 "🔄 刷新" 重新加载

## 🎓 进阶使用

### 自定义延迟时间

编辑 `config.py`:

```python
AUTO_FILL_DELAY = 2000  # 改为 2000 毫秒（2秒）
```

### 备份数据

```bash
cp data/auto_form_filler.db backup/$(date +%Y%m%d).db
```

### 查看数据库

```bash
sqlite3 data/auto_form_filler.db
.tables  # 查看所有表
select * from cards;  # 查看名片
```
