# 🚀 腾讯文档表单 - 快速开始

## 💡 5分钟上手指南

### 第 1 步：启动应用

```bash
cd /Users/chenchen/Desktop/开发项目/auto-form-filler-py
source venv/bin/activate
python main.py
```

### 第 2 步：配置名片

1. 点击 **"📇 名片管理"**
2. 选择 **"名片1"** 或创建新名片
3. 添加配置项（根据表单问题配置）

**示例配置**（基于提供的表单）:
```
手机号【着急时联系】 = 13800138000
微信 = weixin123
抖音昵称 = 测试账号
抖音账号 = test123456
粉丝数量 = 10000
主页链接 = https://example.com/profile
赞藏量 = 5000
所在地 = 北京
（自行报价） = 300
账号类型 = 母婴
```

### 第 3 步：添加表单链接

1. 点击 **"🔗 链接管理"**
2. 点击 **"➕ 新增链接"**
3. 输入信息：
   - **名称**: `抖音-Vitavea白加黑胶囊`
   - **URL**: `https://docs.qq.com/form/page/DV0JwTG9BTmJIZWNr`
   - **状态**: 启用
4. 保存

### 第 4 步：自动填写

1. 点击 **"📝 自动填写"**
2. 在左侧选择 **名片**
3. 在左侧选择 **链接**
4. 点击 **"✏️ 自动填写"** 按钮
5. 等待 3-5 秒
6. 查看结果 ✅

## 🔍 如果填写失败

### 方法 1：使用诊断功能

1. 点击工具栏的 **"🔍 诊断页面"**
2. 查看弹窗显示的问题列表
3. 在控制台查看详细信息
4. 对比问题标题和名片字段名
5. 调整名片配置

### 方法 2：查看控制台日志

在终端中会输出详细日志：
```
====== 🚀 开始填写腾讯文档表单 ======
⏳ 等待页面加载...
✅ 页面已加载，找到 10 个问题

  ✅ 填写成功
...
```

### 方法 3：调整配置

如果某个字段没有填写成功：
1. 查看诊断结果中的**准确问题标题**
2. 在名片配置中使用**完全相同的字段名**
3. 或使用**包含关键词**的字段名

## 🎯 实用技巧

### 技巧 1：批量配置

一次性配置好所有字段，下次直接使用：
- 创建多个名片对应不同账号
- 每个名片保存常用的配置
- 切换名片即可切换配置

### 技巧 2：快速查看配置

在自动填写页面：
- 点击 **"📋 配置面板"** 查看当前名片的所有配置
- 如果自动填写失败，可以参考配置手动填写

### 技巧 3：使用关闭按钮

填写完成后：
- 点击左侧面板的 **"❌ 关闭窗口"**
- 或点击顶部工具栏的 **"⬅️ 返回"**
- 快速返回主界面

## ❓ 常见问题

**Q: 为什么有些字段没填上？**
- A: 字段名可能不匹配，使用诊断功能查看准确的问题标题

**Q: 提示"需要登录"怎么办？**
- A: 在 WebView 中先登录腾讯文档账号

**Q: 填写速度太慢？**
- A: 已优化到 3-5 秒，如需更快可修改 `config.py` 中的延迟

**Q: 如何备份配置？**
- A: 复制 `data/auto_form_filler.db` 文件即可
